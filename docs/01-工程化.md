# 工程化

## Maven vs Gradle

- **Maven**：配置文件 `pom.xml`，老牌主流，教程最多
- **Gradle**：配置文件 `build.gradle`，灵活，Android 和大型项目用

## pom.xml — Maven 项目配置文件

### 项目信息

```xml
<groupId>com.example</groupId>
<artifactId>springboot-learning</artifactId>
<version>0.0.1-SNAPSHOT</version>   <!-- SNAPSHOT = 开发中 -->
```

### 依赖

```xml
<dependencies>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-web</artifactId>
    </dependency>
</dependencies>
```

Maven 按 `<groupId>:<artifactId>:<version>` 坐标去中央仓库自动下载 jar 包。

**starter 的含义**：一个启动器 = 一组相关 jar 包。比如 `starter-web` 包含了 Spring MVC、内嵌 Tomcat、Jackson，不用逐个声明。

### 父 POM

```xml
<parent>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-parent</artifactId>
    <version>3.5.14</version>
</parent>
```

继承了父 POM 的版本号管理，依赖可以省略 `<version>`，避免版本冲突。

### 构建插件

```xml
<build>
    <plugins>
        <plugin>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-maven-plugin</artifactId>
        </plugin>
    </plugins>
</build>
```

把项目打成 Fat Jar（代码 + Tomcat + 所有依赖），一行命令启动：

```bash
java -jar springboot-learning-0.0.1-SNAPSHOT.jar
```

## application.yml vs application.properties

```yaml
# YAML（推荐）
server:
  port: 8080
```

```properties
# Properties
server.port=8080
```

## Lombok — 消灭模板代码

```xml
<dependency>
    <groupId>org.projectlombok</groupId>
    <artifactId>lombok</artifactId>
    <optional>true</optional>
</dependency>
```

IDEA 需装 Lombok 插件。

### 改前 vs 改后

```java
// 改前：手写所有 getter/setter，字段越多越啰嗦
public class User {
    private Long id;
    private String name;
    private String email;

    public Long getId() { return id; }
    public void setId(Long id) { this.id = id; }
    public String getName() { return name; }
    public void setName(String name) { this.name = name; }
    public String getEmail() { return email; }
    public void setEmail(String email) { this.email = email; }
}
```

```java
// 改后：@Data 全搞定
import lombok.Data;

@Data
public class User {
    private Long id;
    private String name;
    private String email;
}
```

常用注解：

| 注解 | 生成内容 |
|------|------|
| `@Data` | `@Getter` + `@Setter` + `@ToString` + `@EqualsAndHashCode` |
| `@AllArgsConstructor` | 全参构造方法 |
| `@NoArgsConstructor` | 无参构造方法 |

Lombok 在编译期生成代码到 `.class` 文件里，源码看不见但运行时存在，Jackson 和 MyBatis-Plus 照常用。
