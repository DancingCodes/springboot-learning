# 项目注解整理

## Spring Boot / Spring MVC

### `@SpringBootApplication`

启动类标记，= `@Configuration` + `@EnableAutoConfiguration` + `@ComponentScan`。

```java
import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class SpringbootLearningApplication {
    public static void main(String[] args) {
        SpringApplication.run(SpringbootLearningApplication.class, args);
    }
}
```

### `@RestController`

= `@Controller` + `@ResponseBody`，方法返回值直接写入响应体（JSON/字符串）。

```java
import org.springframework.web.bind.annotation.RestController;

@RestController
public class UserController {
    // 所有方法的返回值直接写入 Response Body
}
```

### `@RequestMapping`

类级别 URL 前缀。

```java
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
@RequestMapping("/user")
public class UserController { }
```

### `@GetMapping` / `@PostMapping` / `@PutMapping` / `@DeleteMapping`

映射 HTTP 请求方法到 Controller 方法。

```java
import org.springframework.web.bind.annotation.DeleteMapping;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.PathVariable;
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.PutMapping;
import org.springframework.web.bind.annotation.RequestParam;

@GetMapping                   // GET /user?page=1&size=10
public Result<Page<User>> list(@RequestParam(defaultValue = "1") int page,
                               @RequestParam(defaultValue = "10") int size) { ... }

@GetMapping("/{id}")          // GET /user/1
public Result<User> getById(@PathVariable Long id) { ... }

@PostMapping                  // POST /user
public String add(@Valid @RequestBody User user) { ... }

@PutMapping                   // PUT /user
public String update(@Valid @RequestBody User user) { ... }

@DeleteMapping("/{id}")       // DELETE /user/1
public String delete(@PathVariable Long id) { ... }
```

### `@RequestParam` / `@PathVariable` / `@RequestBody`

从请求中提取参数。

```java
import org.springframework.web.bind.annotation.PathVariable;
import org.springframework.web.bind.annotation.RequestBody;
import org.springframework.web.bind.annotation.RequestParam;

// GET /user?page=2&size=20  →  page=2, size=20
public Result list(@RequestParam(defaultValue = "1") int page,
                   @RequestParam(defaultValue = "10") int size) { ... }

// GET /user/5  →  id = 5
public User getById(@PathVariable Long id) { ... }

// POST body: {"name":"张三","email":"zhang@example.com"}
public String add(@RequestBody User user) { ... }
```

`@RequestParam` 的 `defaultValue` 指定缺省值。`@RequestBody` 依赖 Jackson 自动将 JSON 反序列化成 Java 对象。

### `@Valid`

触发 Jakarta Validation 校验，加在 `@RequestBody` 参数前。校验失败抛 `MethodArgumentNotValidException`，方法体不会执行。

```java
import jakarta.validation.Valid;
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestBody;

@PostMapping
public String add(@Valid @RequestBody User user) { ... }
```

### `@Service`

声明 Service 层 Bean。

```java
import org.springframework.stereotype.Service;

@Service
public class UserService {
    // Spring 扫描到后自动 new 并放入容器
}
```

### `@Component`

通用 Bean 注解。`@Service` / `@Repository` / `@Controller` 功能一样，只是语义不同。

```java
import org.springframework.stereotype.Component;

@Component
public class JwtAuthFilter extends OncePerRequestFilter { ... }
```

### `@Configuration` + `@Bean`

`@Configuration` 声明配置类。`@Bean` 用在方法上，将返回值注册为 Bean。

```java
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.security.crypto.bcrypt.BCryptPasswordEncoder;
import org.springframework.security.crypto.password.PasswordEncoder;

@Configuration
public class SecurityConfig {

    @Bean
    public PasswordEncoder passwordEncoder() {
        return new BCryptPasswordEncoder();
    }
}
```

和 `@Component` 的区别：`@Component` 加在自己写的类上，`@Bean` 用于需要手动构造的外部类。

### `@Value`

从配置文件注入值，`${key:默认值}` 语法引用 application.yml 中的配置。

```java
import org.springframework.beans.factory.annotation.Value;

public FileController(@Value("${file.upload-dir:./uploads}") String uploadDir) {
    // application.yml: file.upload-dir: ./uploads
}
```

### `@RestControllerAdvice` + `@ExceptionHandler` + `@ResponseStatus`

全局异常拦截器，对所有 Controller 生效。

```java
import org.springframework.http.HttpStatus;
import org.springframework.web.bind.MethodArgumentNotValidException;
import org.springframework.web.bind.annotation.ExceptionHandler;
import org.springframework.web.bind.annotation.ResponseStatus;
import org.springframework.web.bind.annotation.RestControllerAdvice;

@RestControllerAdvice
public class GlobalExceptionHandler {

    @ExceptionHandler(MethodArgumentNotValidException.class)
    @ResponseStatus(HttpStatus.BAD_REQUEST)
    public Result<Void> handleValidation(MethodArgumentNotValidException ex) {
        String msg = ex.getBindingResult().getFieldErrors().stream()
                .map(e -> e.getField() + "：" + e.getDefaultMessage())
                .collect(Collectors.joining("；"));
        return Result.error(400, msg);
    }

    @ExceptionHandler(Exception.class)
    @ResponseStatus(HttpStatus.INTERNAL_SERVER_ERROR)
    public Result<Void> handleOtherException(Exception e) {
        return Result.error(500, e.getMessage());
    }
}
```

Spring 按从具体到模糊匹配 `@ExceptionHandler`：先精确匹配，最后 `Exception.class` 兜底。

### `@Transactional`

开启数据库事务。方法内任意位置抛 `RuntimeException` 时，已执行的 SQL 自动回滚。

```java
import org.springframework.stereotype.Service;
import org.springframework.transaction.annotation.Transactional;
import java.math.BigDecimal;

@Service
public class AccountService {

    private final AccountMapper accountMapper;

    public AccountService(AccountMapper accountMapper) {
        this.accountMapper = accountMapper;
    }

    @Transactional
    public void transfer(Long fromId, Long toId, BigDecimal amount) {
        int rows = accountMapper.deduct(fromId, amount);
        if (rows == 0) {
            throw new IllegalArgumentException("余额不足");  // 抛异常 → 回滚
        }
        int rows2 = accountMapper.addBalance(toId, amount);
        if (rows2 == 0) {
            throw new IllegalArgumentException("目标账户不存在"); // 抛异常 → 第 1 步也撤销
        }
    }
}
```

## MyBatis / MyBatis-Plus

### `@Mapper`

声明 MyBatis Mapper 接口。继承 `BaseMapper<T>` 后自带 CRUD 方法。

```java
import com.baomidou.mybatisplus.core.mapper.BaseMapper;
import org.apache.ibatis.annotations.Mapper;

@Mapper
public interface UserMapper extends BaseMapper<User> {
    // 零 SQL，自带 selectList / selectById / insert / updateById / deleteById
}
```

### `@TableName` + `@TableId` + `@TableField`

实体类到数据库表的映射。

```java
import com.baomidou.mybatisplus.annotation.IdType;
import com.baomidou.mybatisplus.annotation.TableField;
import com.baomidou.mybatisplus.annotation.TableId;
import com.baomidou.mybatisplus.annotation.TableName;
import java.math.BigDecimal;
import java.time.LocalDateTime;

@TableName("user")
public class User {

    @TableId(type = IdType.AUTO)  // 数据库自增主键
    private Long id;
    private String name;
    private String email;
    private Integer age;
    private LocalDateTime createTime;

    @TableField(exist = false)    // 数据库里没有这个字段，忽略映射
    private BigDecimal balance;   // 用来承载关联查询的临时数据
}
```

### `@Update` + `@Param`

自定义 UPDATE SQL，用于 BaseMapper 覆盖不了的场景（如基于当前值加减）。

```java
import com.baomidou.mybatisplus.core.mapper.BaseMapper;
import org.apache.ibatis.annotations.Mapper;
import org.apache.ibatis.annotations.Param;
import org.apache.ibatis.annotations.Update;
import java.math.BigDecimal;

@Mapper
public interface AccountMapper extends BaseMapper<Account> {

    @Update("UPDATE account SET balance = balance - #{amount} WHERE id = #{id} AND balance >= #{amount}")
    int deduct(@Param("id") Long id, @Param("amount") BigDecimal amount);

    @Update("UPDATE account SET balance = balance + #{amount} WHERE id = #{id}")
    int addBalance(@Param("id") Long id, @Param("amount") BigDecimal amount);
}
```

## Lombok

Lombok 在**编译期**生成代码到 `.class` 文件中，源码里看不见，但运行时存在。

### `@Data`

= `@Getter` + `@Setter` + `@ToString` + `@EqualsAndHashCode` + `@RequiredArgsConstructor`。

```java
import com.baomidou.mybatisplus.annotation.TableName;
import jakarta.validation.constraints.Email;
import jakarta.validation.constraints.NotBlank;
import lombok.Data;

@Data
@TableName("user")
public class User {
    private Long id;

    @NotBlank(message = "姓名不能为空")
    private String name;

    @Email(message = "邮箱格式不正确")
    private String email;

    // 不用手写 getter/setter
}
```

### `@RequiredArgsConstructor`

为 `final` 字段生成构造方法，替代 `@Autowired` 字段注入。

```java
import lombok.RequiredArgsConstructor;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
@RequestMapping("/user")
@RequiredArgsConstructor
public class UserController {
    private final UserService userService;
    // Lombok 生成: public UserController(UserService userService) { ... }
    // Spring 自动用这个构造器注入
}
```

### `@AllArgsConstructor` / `@NoArgsConstructor`

生成全参 / 无参构造方法。通常一起用在 DTO 上。

```java
import lombok.AllArgsConstructor;
import lombok.Data;
import lombok.NoArgsConstructor;

@Data
@NoArgsConstructor
@AllArgsConstructor
public class Result<T> {
    private int code;
    private String msg;
    private T data;
}
```

### `@Slf4j`

自动生成 `log` 静态字段，直接写 `log.info(...)`。

```java
import lombok.extern.slf4j.Slf4j;

@Slf4j
public class SomeService {
    public void doSomething() {
        log.info("处理完成");  // log 由 @Slf4j 自动生成
    }
}
```

## Jakarta Validation（参数校验）

Spring Boot 3.x 使用 Jakarta EE 9+，包是 `jakarta.validation.constraints.*`（不是老的 `javax.validation.*`）。

校验注解加在实体类字段上，配合 Controller 方法的 `@Valid` 使用。

| 注解 | 作用 | 示例 |
|------|------|------|
| `@NotBlank` | 字符串不能为 null 或全空白 | `@NotBlank(message = "姓名不能为空")` |
| `@Email` | 必须是合法邮箱格式 | `@Email(message = "邮箱格式不正确")` |
| `@Min` / `@Max` | 整数范围 | `@Min(0) @Max(150)` |
| `@NotNull` | 不能为 null（用于 Long、BigDecimal） | `@NotNull(message = "金额不能为空")` |
| `@DecimalMin` | BigDecimal 最小值 | `@DecimalMin(value = "0.01", message = "金额必须大于0")` |

```java
import jakarta.validation.constraints.DecimalMin;
import jakarta.validation.constraints.Email;
import jakarta.validation.constraints.Max;
import jakarta.validation.constraints.Min;
import jakarta.validation.constraints.NotBlank;
import jakarta.validation.constraints.NotNull;
import java.math.BigDecimal;

public class CreateUserRequest {

    @NotBlank(message = "姓名不能为空")
    private String name;

    @Email(message = "邮箱格式不正确")
    private String email;

    @Min(value = 0, message = "年龄不能为负数")
    @Max(value = 150, message = "年龄超出合理范围")
    private Integer age;

    @NotNull(message = "金额不能为空")
    @DecimalMin(value = "0.01", message = "金额必须大于0")
    private BigDecimal amount;
}
```

## Spring AOP

### `@Aspect` + `@Pointcut` + `@Around`

声明切面、定义切入点、环绕通知。

```java
import lombok.extern.slf4j.Slf4j;
import org.aspectj.lang.ProceedingJoinPoint;
import org.aspectj.lang.annotation.Around;
import org.aspectj.lang.annotation.Aspect;
import org.aspectj.lang.annotation.Pointcut;
import org.springframework.stereotype.Component;

@Slf4j
@Aspect
@Component  // 切面本身也要是 Bean
public class ControllerLogAspect {

    @Pointcut("execution(* com.example.springbootlearning.controller.*.*(..))")
    public void controllerMethods() {}

    @Around("controllerMethods()")
    public Object logAround(ProceedingJoinPoint joinPoint) throws Throwable {
        String method = joinPoint.getSignature().toShortString();
        log.info("→ {} | 参数: {}", method, joinPoint.getArgs());
        long start = System.currentTimeMillis();
        Object result = joinPoint.proceed();          // 必须调用 proceed() 才会执行原方法
        log.info("← {} | 耗时: {}ms", method, System.currentTimeMillis() - start);
        return result;                                // 必须返回结果
    }
}
```

关键约束：
- 切面只对 Spring Bean 生效，自己 `new` 的对象不走 AOP
- 同一个类内方法互调不走 AOP（`this` 绕过了代理）
- `@Around` 不调 `proceed()` 或不返回其结果，原方法不会执行

## Spring Security

### `@Configuration` + `@EnableWebSecurity`

Spring Security 配置类（Spring Boot 3.x 写法，替代已废弃的 `WebSecurityConfigurerAdapter`）。

三个核心 Bean：

| Bean | 职责 |
|------|------|
| `PasswordEncoder` | 密码加密 |
| `UserDetailsService` | 加载用户 |
| `SecurityFilterChain` | 拦截规则 |

```java
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.security.config.annotation.web.builders.HttpSecurity;
import org.springframework.security.config.annotation.web.configuration.EnableWebSecurity;
import org.springframework.security.config.http.SessionCreationPolicy;
import org.springframework.security.web.SecurityFilterChain;

@Configuration
@EnableWebSecurity
public class SecurityConfig {

    @Bean
    public SecurityFilterChain securityFilterChain(HttpSecurity http) throws Exception {
        http
            .authorizeHttpRequests(auth -> auth
                .requestMatchers("/auth/login").permitAll()
                .anyRequest().authenticated()
            )
            .sessionManagement(session ->
                session.sessionCreationPolicy(SessionCreationPolicy.STATELESS)  // JWT 模式必须无状态
            )
            .csrf(csrf -> csrf.disable());
        return http.build();
    }
}
```

### `@EnableMethodSecurity`

开启方法级权限控制。

```java
import org.springframework.security.access.prepost.PreAuthorize;

@PreAuthorize("hasRole('ADMIN')")
@DeleteMapping("/{id}")
public Result<Void> delete(@PathVariable Long id) { ... }
```

### JWT 流程

```
POST /auth/login (公开) → 验证密码 → 返回 JWT
   ↓
后续请求带 Authorization: Bearer <token>
   ↓
JwtAuthFilter (OncePerRequestFilter) → 解析 token → 注入 SecurityContext
```

- 登录接口必须 `.permitAll()`
- `addFilterBefore(jwtAuthFilter, UsernamePasswordAuthenticationFilter.class)` 把 JWT 过滤器插到认证链前面
- token 过期或签名不对 → `SecurityContext` 为空 → 返回 403

## Knife4j / OpenAPI

### `@Tag` / `@Operation` / `@Parameter` / `@Schema`

```java
import io.swagger.v3.oas.annotations.Operation;
import io.swagger.v3.oas.annotations.Parameter;
import io.swagger.v3.oas.annotations.media.Schema;
import io.swagger.v3.oas.annotations.tags.Tag;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestParam;

@Tag(name = "用户管理")
@RestController
public class UserController {

    @Operation(summary = "分页查询用户")
    @GetMapping
    public Result<Page<User>> list(
        @Parameter(description = "页码") @RequestParam(defaultValue = "1") int page,
        @Parameter(description = "每页条数") @RequestParam(defaultValue = "10") int size) { ... }
}
```

```java
@Schema(description = "用户")
public class User {

    @Schema(description = "用户ID")
    private Long id;

    @Schema(description = "姓名")
    private String name;
}
```

```java
import io.swagger.v3.oas.annotations.Hidden;

@RestControllerAdvice

```
