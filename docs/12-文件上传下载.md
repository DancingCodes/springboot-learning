# 文件上传下载

## 上传

```java
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestParam;
import org.springframework.web.multipart.MultipartFile;
import java.io.IOException;
import java.util.UUID;

@PostMapping("/upload")
public Result<FileVO> upload(@RequestParam("file") MultipartFile file) throws IOException {
    String storedName = UUID.randomUUID() + "_" + file.getOriginalFilename();
    file.transferTo(uploadDir.resolve(storedName).toFile());
    FileVO vo = new FileVO();
    vo.setOriginalName(file.getOriginalFilename());
    vo.setStoredName(storedName);
    vo.setSize(file.getSize());
    return Result.success(vo);
}
```

返回 `FileVO`（originalName、storedName、size），统一响应格式。

流程：`MultipartFile` 接收 → `UUID_原始名` 防重名 → `transferTo()` 写磁盘。

## 下载

```java
import org.springframework.core.io.FileSystemResource;
import org.springframework.core.io.Resource;
import org.springframework.http.HttpHeaders;
import org.springframework.http.MediaType;
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestParam;
import java.net.URLEncoder;
import java.nio.charset.StandardCharsets;

@GetMapping("/download")
public ResponseEntity<Resource> download(
        @RequestParam String storedName,
        @RequestParam(defaultValue = "false") boolean download) {

    Resource resource = new FileSystemResource(uploadDir.resolve(storedName));
    if (!resource.exists()) {
        return ResponseEntity.notFound().build();
    }

    String originalName = storedName.substring(storedName.indexOf("_") + 1);
    String disposition = download
            ? "attachment; filename*=UTF-8''" + URLEncoder.encode(originalName, StandardCharsets.UTF_8)
            : "inline";

    return ResponseEntity.ok()
            .contentType(download ? MediaType.APPLICATION_OCTET_STREAM : MediaType.IMAGE_JPEG)
            .header(HttpHeaders.CONTENT_DISPOSITION, disposition)
            .body(resource);
}
```

`?download` 参数控制浏览器行为：

| 请求 | Content-Disposition | 效果 |
|------|---------------------|------|
| `/download?storedName=xxx.jpg` | `inline` | 浏览器内直接展示 |
| `/download?storedName=xxx.jpg&download=true` | `attachment` | 弹出下载框 |

`filename*=UTF-8''` + `URLEncoder.encode` 保证中文文件名不乱码（RFC 5987）。

## 配置

```yaml
spring:
  servlet:
    multipart:
      max-file-size: 10MB
      max-request-size: 10MB

file:
  upload-dir: ./uploads
```

## 安全放行

图片通过 `<img>` 标签加载不带 JWT token，需要放行下载路径：

```java
.requestMatchers("/file/download").permitAll()
```

## 核心类

| 类 | 作用 | 常用方法 |
|------|------|------|
| `MultipartFile` | 上传文件封装 | `getOriginalFilename()`, `transferTo()`, `getSize()` |
| `Resource` | 资源抽象 | `exists()`, `getInputStream()` |
| `ResponseEntity<Resource>` | 文件下载标准返回类型 | 控制状态码、响应头、响应体 |
